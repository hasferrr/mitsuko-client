# Update Sidebar Dropdown to Match Project/Batch Page Style

## Context
The sidebar dropdown (`app-sidebar-projects.tsx`) currently has Export and Delete items with inconsistent styling vs the project/batch page dropdowns. It's also missing the Archive option.

## Changes

### 1. Add Archive dropdown item
Add an Archive/Unarchive item to the sidebar dropdown, matching the pattern used in project/batch cards:
- Non-archived projects → show "Archive" with `Archive` icon
- Archived projects → show "Unarchive" with `ArchiveRestore` icon
- Requires destructuring `handleArchive` from `useProjectActions()`

### 2. Rename labels to match
- "Export Project" → "Export"
- "Delete Project" → "Delete"

### 3. Match icon styling
Current sidebar icons use `className="text-muted-foreground"`, while project/batch cards use `className="size-4"`. Update sidebar icons to `className="size-4"` to match. The delete item should also get `className="text-destructive"` to match.

### 4. Add imports
Add `Archive`, `ArchiveRestore`, `Trash` to the lucide-react import (replacing `Trash2` with `Trash` for consistency).
